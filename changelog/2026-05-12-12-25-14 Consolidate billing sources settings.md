# Consolidate billing sources settings

API Keys and Subscriptions now live together under a single Billing sources settings section. The page uses expandable list rows for API keys, Claude.ai subscription credentials, and ChatGPT subscription credentials so the layout matches the extendable-list feel used by Presets and Update sources.

The old `/settings/subscriptions` page now redirects to `/settings` so existing links continue to work while users land on the consolidated settings experience.
